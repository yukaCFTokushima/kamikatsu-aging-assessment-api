# 上勝町老化評価API  
*Kamikatsu Aging Assessment API (conceptual specification)*

## 概要
本リポジトリは、徳島県上勝町を対象に、「老い」を個人の問題ではなく、**住宅・福祉・地域支援を横断して捉えるための〈老いのアセスメント〉**という考え方を、**概念API（設計仕様）**として整理したものです。  
本APIは、超高齢化・人口減少が進む中山間地域において、「どのように住み続け、どのように支え合い、どのように最期を迎えるか」を、データ構造として可視化・共有することを目的としています。

本リポジトリは **アーバンデータチャレンジ2025（UDC2025）応募作品の説明資料の一部**として作成されています。

---

## 背景と問題意識
上勝町では、高齢化率の上昇により、  
- 独居・高齢世帯の増加  
- 医療・介護・福祉サービスの分散  
- 住み慣れた家で暮らし続けることの困難さ  

といった課題が複合的に進行しています。

一方で、制度上のデータは行政や医療機関ごとに分断されており、  
**「生活としての老い」**を横断的に捉える共通の枠組みは存在していません。

本構想では、上勝町社会福祉協議会の現場知見を起点に、  
「老いの状態をどう把握し、誰が・どのタイミングで・どう関わるか」を  
**データ設計の言葉で整理し直す**ことを試みています。

---

## 老いのアセスメントという考え方
本APIが扱う「老いのアセスメント」とは、  
医療データや介護認定情報そのものではなく、

- 住環境の変化（段差、距離、移動負荷）
- 生活リズムの変化
- 支援が“必要になる前兆”
- 支援が入った／入らなかった結果

といった **生活・住宅・支援の関係性** を、  
個人を特定しない形で構造化する視点です。

これは、防災分野における「個別避難計画」や「災害アセスメント」の考え方を、  
平時の〈老い〉に応用したものでもあります。

---

## APIの位置づけ
本リポジトリに記載されているAPIは、以下の特徴を持ちます。

- 実装済みのAPIではなく、**概念設計・仕様レベル**
- 個人情報・生データ・要配慮個人情報は一切含まない
- 統計化・集約化されたデータ構造のみを想定
- 住宅設計、地域支援設計、政策検討に利用可能な粒度

APIは、  
**「住宅 × 支援 × 老いの状態」**をつなぐ中間層として位置づけています。

- 📘 APIメニュー一覧：老いのアセスメントAPIの全体構造と想定メニューは [docs/api/api-menu.md](docs/api/api-menu.md) を参照

---

## APIメニュー一覧（概念設計）
| API名 | 概要 | 主な項目（例） | 想定される利用 | 備考 |
|---|---|---|---|---|
| aging-assessment-summary | 高齢期の生活状態を俯瞰的に把握する総合指標 | 生活自立度、住環境適合度、支援接続度 | 住宅設計・地域支援配置の起点 | 個人非特定・集約指標 |
| housing-fit-index | 住環境と老いの状態の適合度を評価 | 段差有無、動線距離、立地条件 | 改修・集住・長屋設計検討 | 建築・住宅政策向け |
| support-connection-map | 支援との接続状況を構造的に把握 | 支援接触頻度、支援種別 | 支援の偏在・空白の可視化 | 社協・包括の連携設計 |
| social-engagement-index | 社会的関与の状態を把握 | 交流機会、居場所参加 | 孤立予防・雑談導線設計 | 見守りの前段階 |
| risk-prevention-signal | 支援が必要になる前兆の把握 | 生活変化の累積傾向 | 予防的介入検討 | 早期支援の判断材料 |

## 想定される活用例
- 集住・長屋型住宅の設計検討
- 高齢者向け住宅の見守り設計
- 社協・包括支援センター・自治体間の役割整理
- 他自治体への横展開時の共通設計図
- 視察・研修・研究用途での参照モデル

本APIは、上勝町固有の事情から出発していますが、  
**20年後の多くの基礎自治体が直面する状況に共通する設計課題**を扱っています。

---

## データに関する注意
- 本リポジトリには、**実データ・個人データ・行政保有データは含まれていません**
- APIはすべて仮想・概念仕様です
- 実装時には、各自治体・関係機関の合意形成と法令遵守が前提となります

---

## 関係者・構想背景
- 上勝町社会福祉協議会（現場知見）
- 大学研究者による評価・設計視点
- Code for Tokushima による構想設計・可視化支援

「制度は守るもの、生活はつなぐもの」という現場の視点を、  
データと設計の言葉に翻訳する試みです。
なお、本APIは実装を目的としたものではなく、地域福祉・住宅・政策検討のための共通言語としての設計図を提示するものです。

## About this repository (English)
This repository contains **conceptual API specifications** for an “Aging Assessment” framework to support housing, welfare, and community-based care design in Kamikatsu Town, Japan.  
It is prepared as part of an **Urban Data Challenge 2025** submission and includes **no personal or raw data**.
