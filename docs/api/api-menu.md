# APIメニュー一覧（概念仕様）

本ドキュメントは、「上勝町老化評価API」において想定している  
**〈老いのアセスメント〉を住宅・支援・地域設計に接続するための概念的APIメニュー一覧**です。

本APIは**実装を目的としたものではなく**、  
自治体・社協・研究者・設計者が共通の視点で議論するための  
**設計仕様（共通言語）**として整理されています。

※ 個人情報・生データ・行政保有データは一切含みません。

---

## API一覧（Conceptual Endpoints）

### 1. aging-assessment-summary
**老いの状態を、生活・住宅・社会関係の観点から俯瞰的に整理するための基本アセスメントAPI。**

- 対象：地域単位／集計単位（個人単位は想定しない）
- 主な観点：
  - 生活変化の兆候
  - 支援が入る前後の状態差
  - 社会的接続の有無
- 想定用途：
  - 地域全体の老いの傾向把握
  - 住宅・支援設計の前提整理

---

### 2. housing-fit-index
**住環境が高齢期の生活にどの程度適合しているかを構造的に整理するためのAPI。**

- 主な観点：
  - 段差・動線・移動距離
  - 改修余地
  - 集住・長屋化の可能性
- 想定用途：
  - 高齢者向け住宅設計
  - 空き家活用・集住モデル検討
  - 「住み続けられるか」の判断材料

---
