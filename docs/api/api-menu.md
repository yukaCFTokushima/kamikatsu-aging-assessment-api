# APIメニュー一覧（概念）

本ドキュメントは、「上勝町老化評価API」において想定している
概念的なAPIメニューの一覧です。
すべて実装を前提としない設計仕様であり、個人データは含みません。

---

## API一覧

### 1. aging-assessment-summary
老いの状態を、生活・住宅・社会関係の観点から俯瞰的に整理するための
基本アセスメントAPI。

- 対象：地域単位・集計単位
- 主な観点：生活変化／支援前兆／社会的接続

---

### 2. housing-fit-index
住環境が高齢期の生活にどの程度適合しているかを
構造的に評価するためのAPI。

- 観点：段差、動線、距離、改修余地、集住可能性
- 用途：住宅設計・改修検討

---

### 3. support-connection-map
社協・包括・医療・見守り等の支援が
どのように配置・接続されているかを可視化するAPI。

- 観点：支援密度、役割分担、空白エリア
- 用途：支援設計・政策検討

---

### 4. early-signals-of-aging
支援が「必要になる前」の変化を捉えるための
前兆的アセスメントAPI。

- 観点：生活頻度低下、外出減少、雑談接触の減少
- 用途：予防的介入検討
